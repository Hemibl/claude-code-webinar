# Claude Code Webinar - Product Ops Community

## About This Project
וובינר חינמי בן 90 דקות ללימוד Claude Code לאנשי Product Ops.
הקהל: כ-25 אנשי ProdOps מקבוצת ווצאפ, רמה טכנית לא ידועה, לא מכירים את חמי.

## Project Structure
- `/demos/` - חומרי דמו (בריף פיקטיבי, דשבורד מוגמר)
- `/materials/` - מדריך התקנה, cheat sheet למשתתפים
- `/forms/` - טופס הרשמה
- `/assets/` - תמונות, לוגו, ויזואלים

## Webinar Structure (90 min)
1. **(10 דק') פתיחה** - "אני לא מפתח, בדיוק כמוכם"
2. **(10 דק') וואו ראשון** - דמו חי: סידור תיקייה / ניתוח קבצים
3. **(10 דק') הבסיס** - תיקייה=workspace, CLAUDE.md, קונטקסט + טיפים פרקטיים
4. **(25-30 דק') דמו מרכזי חי** - בריף/דרישות → אפיון + פרוטוטייפ (מאפס)
5. **(10-15 דק') דמו מוגמר** - דשבורד מנתונים גולמיים (מראים תוצאה)
6. **(15 דק') סיום + Q&A** - מה עוד אפשר, איך ממשיכים

## Practical Tips to Cover
- CLAUDE.md - איך מלמדים את הסוכן מי אתה
- מודלים שונים (Opus, Sonnet, Haiku)
- skip-permissions - דילוג על אישורים
- טרמינלים מקבילים - עבודה על כמה משימות
- סקילים - יכולות ארוזות
- תוסף Chrome - אוטומציית דפדפן
- תוסף VS Code - אזכור קצר

## Key Principle
הדמו החי חייב להיראות ספונטני אבל מתוכנן היטב. לתרגל לפחות 3 פעמים לפני המפגש.

## Strategic Goal
בניית שם ונטוורקינג בקהילת ProdOps. חיבור לפוזישנינג כ-AI Strategy Advisor.
