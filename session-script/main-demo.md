# דמו מרכזי (25-30 דקות)

> Source: Deep Interview, 2026-03-24

## כיוון
חמי הכי בטוח עם: בריף → מסמך אפיון → פרוטוטייפ → מצגת.
כבר עשה את זה בעבר ומרגיש בנוח.

## דילמה
הדמו צריך לענות על הצרכים של המשתתפים (מעקב ידני, סנכרון צוותים, דוחות) -
לא רק להראות משהו שנוח לחמי.

## כיוון שנבחר: שילוב של כמה דברים
פלואו שלם שמראה את הכוח של Claude Code מקצה לקצה:
בריף קצר → מסמך אפיון → פרוטוטייפ/דשבורד → מצגת
