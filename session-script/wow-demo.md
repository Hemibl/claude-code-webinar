# וואו ראשון - דמו פשוט (10 דקות)

> Source: Deep Interview, 2026-03-24

## כיוון
סידור תיקיית Downloads האמיתית של חמי - משהו שכולם מכירים ומבינים מיד.

## תכנון
1. **הכנה מראש:** לפרק את הקבצים מהתיקיות המסודרות ולזרוק את הכל ברמה אחת בתיקיית ההורדות - לבלגן מכוון
2. **במפגש:** להראות את הבלגן, לפתוח Claude Code, ולבקש ממנו לסדר
3. **התוצאה:** תיקיות מסודרות לפי סוג/נושא תוך שניות

## Key Points
- קבצים אמיתיים - לא מוכן מראש, נראה אותנטי
- כולם מכירים את הכאב של Downloads מבולגנת
- מראה את היכולת הבסיסית: Claude Code עובד על הקבצים שלך
- פשוט להבנה: אפילו מי שלא מכיר CLI מבין "סדר לי את התיקייה"
