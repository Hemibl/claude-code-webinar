מדריך התקנה - Claude Code

הכנה למפגש רביעי 25.3 | 21:00

היי,
כדי שנוכל לנצל את הזמן שלנו במפגש ולצלול ישר לעבודה בלי לבזבז דקות יקרות על התקנה טכנית, אשמח אם תוכלו להתקין את Claude Code אצלכם במחשב לפני המפגש.

המדריך קצר ופשוט.

אז מה צריך לעשות לקראת המפגש?

1

מנוי Claude פעיל

נרשמים באתר claude.ai ורוכשים מנוי Pro (עולה $20 לחודש). חובה לפני המפגש.

2

התקנת Claude Code

בחרו את מערכת ההפעלה שלכם למטה ועקבו אחרי ההוראות:

1

פתחו את Terminal

לחצו Cmd+Space (מקש הפקודה + רווח), כתבו Terminal ולחצו Enter

2

התקינו את Claude Code

העתיקו והדביקו את הפקודה הבאה בטרמינל ולחצו Enter:

Terminal curl -fsSL https://claude.ai/install.sh | bash
אחרי שההתקנה מסתיימת, סגרו את הטרמינל ופתחו חלון חדש.
3

סגרו ופתחו מחדש את Terminal

אחרי ההתקנה, סגרו את חלון הטרמינל ופתחו חלון חדש.

4

בדקו שזה עובד

בחלון החדש, כתבו ולחצו Enter:

Terminal claude
5

התחברו לחשבון

ייפתח דפדפן באופן אוטומטי - התחברו לחשבון Claude שלכם ועקבו אחרי ההוראות על המסך

1

התקינו Git for Windows

גשו לאתר git-scm.com והורידו את Git for Windows. התקינו עם הגדרות ברירת המחדל (פשוט לחצו Next עד הסוף).

2

פתחו את PowerShell

לחצו על תפריט Start, חפשו PowerShell ופתחו אותו (לא צריך הרשאות Administrator)

3

התקינו את Claude Code

העתיקו והדביקו את הפקודה הבאה ב-PowerShell ולחצו Enter:

PowerShell irm https://claude.ai/install.ps1 | iex
אחרי שההתקנה מסתיימת, סגרו את PowerShell ופתחו חלון חדש.
4

סגרו ופתחו מחדש את PowerShell

חשוב! אחרי ההתקנה צריך לסגור את חלון PowerShell ולפתוח חלון חדש.

5

בדקו שזה עובד

בחלון PowerShell החדש, כתבו ולחצו Enter:

PowerShell claude
6

התחברו לחשבון

ייפתח דפדפן באופן אוטומטי - התחברו לחשבון Claude שלכם ועקבו אחרי ההוראות על המסך

איך יודעים שזה עובד?

אחרי שמריצים claude בטרמינל, אמור להיפתח ממשק שנראה בערך ככה:

אם אתם רואים משהו דומה - ההתקנה הצליחה. מוכנים למפגש!

⚠ בעיות נפוצות - Mac

סגרו את כל חלונות הטרמינל ופתחו מחדש. אם לא עוזר - התקינו מחדש.
הוסיפו sudo לפני הפקודה:
sudo curl -fsSL https://claude.ai/install.sh | bash
חפשו בטרמינל קישור (URL) והעתיקו אותו ידנית לדפדפן.

⚠ בעיות נפוצות - Windows

סגרו את כל חלונות PowerShell ופתחו מחדש. אם לא עוזר - התקינו מחדש.
פתחו PowerShell כ-Administrator (קליק ימני על PowerShell ובחרו "Run as Administrator") ונסו שוב.
הריצו את הפקודה הבאה ב-PowerShell:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
אשרו עם Y ונסו שוב.
חפשו בטרמינל קישור (URL) והעתיקו אותו ידנית לדפדפן.
